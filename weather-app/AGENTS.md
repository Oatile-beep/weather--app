# Weather App

A responsive single-page weather application built with vanilla JavaScript, CSS, and HTML.

## Technical Stack
- **HTML5** - Semantic markup
- **CSS3** - Responsive design with CSS Grid, Flexbox, animations
- **JavaScript (ES6+)** - Async/await, fetch API, DOM manipulation
- **OpenWeatherMap API** - Weather data source

## Key Features Implemented
- Async API calls with error handling
- Geolocation integration
- 5-day weather forecast
- Responsive mobile-first design
- Loading states and error messages
- Accessible ARIA labels

## API Configuration
API key is pre-configured. To use your own key, edit `index.html` line 74:
`const API_KEY = '7684fd5ef96d77c2651a2a11db2aa34e';`
Replace with your OpenWeatherMap API key.

## Project Structure
- Single-file app (index.html) for easy deployment
- No build process required
- Static hosting compatible
